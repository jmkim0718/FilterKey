## 다운로드 전, 이 앱은 관리자 권한이 필요함을 알림
- 만든 사람이 세이람 100개 사서 만든거라 간단하게 토글로해서 메이플하면서 작동시키려면 관리자 권한이 필요했음

## 주의사항
- 기본적으로 필터키는 본인 pc의 레지스트리를 자동으로 수정하는 원리라서 정지와 관련이 없으나, 수치조절로 악용 혹은 매크로와 결합 후 사용 등등으로 정지가 되면 책임지지 않음을 알림. 
- 수년째 사용중이고 정지된적은 없지만 혹여나 정지가 된다해도 책임지지 않음. 사용이 꺼려지면 사용x.
- 건의사항이나 오류 혹은 수정해야하는 사항이 발생하면 쪽지로 부탁

## 다운로드

 **[Releases 페이지](../../releases)** 에서 최신 `MapleFilterKey.exe` 를 받아 더블클릭

- 처음 실행 시 "Windows의 PC 보호" 창이 뜨면 → **추가 정보** → **실행**
  (개인 제작이라 해야하는 과정이고 아무런 악성코드 같은거 숨겨놓지 않았으니 걱정 안하셔도됨)

## 기능

- **프리셋 관리**: 여러 수치 조합을 저장 / 이름 변경 / 삭제
- **수치 편집**: `수정하기` → `저장하기` 
- **전역 단축키**: 메이플 창이 떠 있어도 지정한 키로 즉시 켜기/끄기 (사용자 지정 가능)
- **트레이 상주**, **종료 시 자동 끄기** 옵션

## 사용법

1. 왼쪽에서 프리셋을 **선택** (선택만 되고 켜지지는 않음)
2. 필요하면 `수정하기` 로 수치를 바꾸고 `저장하기`.
3. `이 프리셋으로 켜기` 를 누르면 적용
4. 옵션에서 **켜기/끄기 단축키**를 `설정` 하면 게임중에도 껐다 켰다 가능
